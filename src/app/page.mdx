# Welcome to Your Documentation

This is your documentation homepage built with [Nextra](https://nextra.site/).

## 🚀 Getting Started

Edit this file at `src/app/page.mdx` to customize your homepage.

## ✨ Features

- 🎨 **Beautiful Dark Mode** - Seamless light/dark theme switching
- 📱 **Fully Responsive** - Mobile-first design
- ⚡ **Lightning Fast** - Built on Next.js 15 with Turbopack
- 🔍 **Full-Text Search** - Built-in search powered by Nextra
- 📝 **MDX Support** - Write docs with React components
- 🎯 **SEO Optimized** - Meta tags and sitemap generation

## 📚 Documentation Structure

Organize your documentation in folders:

```
src/app/
├── page.mdx              # Homepage (this file)
├── getting-started/      # Getting started section
│   ├── page.mdx
│   └── installation.mdx
└── guides/              # Guides section
    ├── page.mdx
    └── advanced.mdx
```

## 🎨 Customization

1. **Update Branding**: Edit `theme.config.tsx`
2. **Customize Styles**: Modify `src/app/globals.css`
3. **Add Components**: Create components in `src/components/`
4. **Update Metadata**: Edit `src/app/layout.tsx`
